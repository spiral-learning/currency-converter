Welcome to the Currency Conversion API server.


You probably want to use the API at /convert as shown below.

Example:

https://jitterted-currency-conversion.herokuapp.com/convert?from=USD&to=GBP&amount=100

Query Parameters:

  • from=USD - source currency (only USD supported)
  • to=JPY - target currency (see supported currencies below)
  • amount=100 - the amount (BigDecimal) in the source currency to convert

Supported target currencies are:

  • GBP (British Pound)
  • JPY (Japanese Yen)
  • DOGE (Dogecoin)
  • BTC (Bitcoin)
